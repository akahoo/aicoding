# 掼蛋游戏开发需求

## 项目需求

**项目名称：** game-guandan（掼蛋扑克游戏）

**核心功能：**
1. 4 人标准局（2v2 组队）
2. 无需登录，系统分配昵称，用户可修改
3. 通过邀请 URL 让多个用户进入同一牌局
4. 实时在线打牌（WebSocket 通信）
5. 内存存储牌局数据

**技术栈：**
- 前端：React + TailwindCSS
- 后端：Node.js + Express + Socket.IO
- 存储：内存存储（Room 管理）

**项目结构：**
- client/ - 前端 React 应用
- server/ - 后端 Node.js 服务
  - game/ - 游戏逻辑（deck, rules, room, player）

**游戏功能（优先确保完整性）：**
1. 创建房间（生成邀请 URL）
2. 加入房间（通过 URL）
3. 4 人就位后开始游戏
4. 发牌（每人 27 张，两副牌 108 张）
5. 理牌（按牌型排序）
6. 出牌（选择牌、出牌）
7. 牌型判断（单张、对子、三张、顺子等）
8. 回合制（顺时针出牌）
9. 胜负判定（一方两人先出完牌）
10. 聊天功能
