## 🎮 游戏流程

1. **访问首页** → 输入昵称（或随机生成）
2. **创建房间** → 生成 8 位房间号和邀请 URL
3. **邀请好友** → 复制 URL 发送给 3 位好友
4. **准备开始** → 所有玩家点击"准备"
5. **开始游戏** → 房主点击"开始游戏"
6. **发牌** → 每人 27 张，自动排序
7. **出牌** → 顺时针出牌，选择牌型或"不要"
8. **胜负** → 一方两人先出完牌获胜
