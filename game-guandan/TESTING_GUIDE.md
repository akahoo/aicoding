## 🔍 常见问题排查

### 问题 1：访问邀请链接还是显示首页

**可能原因：**
- URL 格式不正确
- 前端路由配置问题

**解决方法：**
```bash
# 检查 URL 格式
http://localhost:3000/game/{8 位房间号}

# 例如：
http://localhost:3000/game/abc12345
```

---

### 问题 2：加入后房间没反应

**可能原因：**
- Socket 连接未建立
- 后端未广播事件

**检查步骤：**

1. **浏览器控制台**
   ```javascript
   // 应该看到：
   "已连接到服务器"
   "新玩家加入：{玩家昵称}"
   ```

2. **后端日志**
   ```bash
   # 应该看到：
   玩家连接：{socketId}
   玩家 {playerId} 加入房间 {roomId}
   ```

3. **检查 Socket 连接**
   ```javascript
   // 在浏览器控制台运行：
   socket.connected // 应该返回 true
   ```

---

### 问题 3：提示"房间不存在"

**可能原因：**
- 房间号错误
- 房间已被删除
- 后端服务未启动

**解决方法：**
```bash
# 1. 检查后端是否运行
curl http://localhost:3001/api/health

# 2. 检查房间是否存在
curl http://localhost:3001/api/room/{roomId}

# 3. 重启后端
cd server
npm run dev
```
