# Todolist 应用

一个简单易用的待办事项管理工具，帮助你组织日常任务。

## 功能特性

- 添加新的待办事项
- 标记完成/未完成状态
- 删除已完成的任务
- 数据持久化（使用浏览器本地存储）
- 响应式设计，支持移动设备
- 简洁美观的用户界面

## 使用方法

1. 打开 `index.html` 文件即可使用
2. 在输入框中输入待办事项内容
3. 点击"添加"按钮将其加入列表
4. 使用"完成"按钮标记已完成的任务
5. 使用"删除"按钮移除不需要的任务

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6)

## 本地运行

直接在浏览器中打开 `index.html` 即可使用，无需服务器。
